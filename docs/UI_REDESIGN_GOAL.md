# UI Redesign Goal

## 当前目标

重做当前 PySide6 桌面应用的 UI，使其更清晰、专业、稳定，但不重写业务逻辑。

## 本轮只允许做

- 调整主窗口布局
- 调整间距、字号、颜色、圆角
- 优化按钮层级
- 增加状态显示
- 优化卡片、日志区、设置区的视觉结构
- 修复最大化/缩放时布局挤压问题

## 本轮不允许做

- 不重写核心业务逻辑
- 不改变任务执行流程
- 不改变数据结构
- 不改接口协议
- 不大规模重构文件结构
- 不引入重型 UI 框架
- 不一次性重做所有模块

## 成功标准

- 应用能正常启动
- 原有主要功能仍可用
- 主界面层级更清楚
- 最大化和非最大化都不挤压
- 状态信息明显
- 主操作按钮清楚
- 日志区域可滚动且不会撑爆窗口
